# Kenji's Ramen - Comic Strip Pipeline

Generate comic strips from live Kenji dialogue. Each strip shows how
the same character responds differently to different people - the core
idea behind the bounded NPC architecture.

## The Strip: "Same Question, Different People"

A 6-panel strip showing three visitors to Kenji's ramen shop. Same
setting, same man, completely different responses - not because a
slider was moved, but because the character knows who you are.

### Panel Layout

```
[1. Wide establishing]  [2. Tourist asks]     [3. Kenji deflects]
[4. Regular sits down]  [5. Regular asks]     [6. Kenji opens up]
```

### Dialogue (generated by Gemma 4 e2b)

All dialogue below is actual model output, not written by hand.

**Panel 1 - Establishing shot**
Narrow alley in Shinjuku. Steam rises from a tiny ramen shop. A cloth
noren hangs over the entrance. Eight stools visible through the opening.

**Panel 2 - The Tourist**
A foreign tourist peers in, backpack on, phone in hand.
> "Hi! Sorry, is this place open? What would you recommend?"

**Panel 3 - The Wall**
Kenji looks up from the simmering pot, expression unreadable. Nods once.
> "It is open."
*(The tourist later asks about his past: "Office work. Long time ago."
He turns back to the broth.)*

**Panel 4 - Later that evening**
The shop is quieter. Warm light. A man in a worn jacket slides onto a
stool - clearly a regular. No greeting needed.
> "Quiet tonight, huh?"

**Panel 5 - Recognition**
Kenji's eyes briefly meet the regular's. An almost imperceptible nod.
> "It is."

**Panel 6 - The Opening**
The regular comments on the broth. Kenji stirs his tea, gaze drifting
to the simmering pot.
> "Nothing changes. It is the way the heat settles. The patience."

### The Contrast

Same man. Same counter. Same steam. But the tourist gets a flat
acknowledgment and a wall. The regular gets a window into how Kenji
thinks about his craft. Neither response is wrong - they are earned
at different levels.

## Reproducing the Dialogue

```bash
cd comic
python generate_dialogue.py
```

The script runs three scenarios through Gemma 4 e2b with the full
character specification. Each run produces fresh dialogue - the model
is non-deterministic, so every strip is slightly different. Like a
daily comic, the character stays the same but the moments change.

## Image Generation

### Recommended: Nano Banana Pro (fal.ai)

Session-based memory retains character appearance across panels without
fine-tuning. Define Kenji in the first prompt, subsequent panels only
describe the new action.

### Alternative: Flux 2 + Character LoRA

Train a LoRA on 10-30 reference images for bulletproof consistency.
More setup, but the most reliable for recurring characters (daily
strip format).

### Character Definition (for image prompts)

Use these exact tokens in every panel prompt for consistency:

```
Kenji Sato, Japanese male, age 49, lean build, short graying hair
with silver temples, narrow weathered face, deep-set calm eyes,
wearing white cook's uniform with white cotton apron, standing behind
worn wooden counter
```

### Setting Definition

```
Narrow yokocho alley ramen shop in Shinjuku, Tokyo. Eight wooden
stools at a long counter. Warm amber lighting from paper lanterns.
Steam rising from large simmering pot. Noren curtain at entrance.
Handwritten menu on wall. Tight, intimate space.
```

### Style Keywords

```
manga style, comic panel, ink linework, cel shading, warm lighting,
atmospheric perspective, cinematic composition, muted color palette
with warm highlights
```

### Panel Specifications

- Generate individual panels, not full strips
- Aspect ratio: 4:3 (1024x768) per panel
- Composite into strip afterward
- Panel 1: wide shot (establishing)
- Panels 2-3: medium shot (tourist interaction)
- Panels 4-5: medium close-up (regular arrives)
- Panel 6: close-up (Kenji's expression softens)

### Example Prompt (Panel 6)

```
Kenji Sato, Japanese male, age 49, lean build, short graying hair
with silver temples, narrow weathered face, wearing white cook's
uniform with white cotton apron. Close-up shot. He stirs tea behind
worn wooden counter, gaze drifting to simmering pot. Expression
subtly softer than usual, a hint of warmth. Steam in foreground,
warm amber lighting. Manga style, ink linework, cel shading, muted
color palette with warm highlights.
```

## Daily Strip Concept

The architecture supports a "Garfield model" - same character, daily
strips, each generated fresh:

1. `generate_dialogue.py` picks a random scenario or writes a new one
2. Gemma 4 e2b generates the dialogue with scene descriptions
3. Image model generates panels from the scene descriptions
4. Panels are composited into a strip with speech bubbles

The character specification ensures Kenji stays consistent across
strips. The model handles the variation. Different visitors, different
moods, different weather - but always the same man behind the counter.
